Build Something

You're ready. Here's how to actually do it.

1

Start every project the same way

  1. Create a new folder inside projects and name it after your project
  2. Open VS Code → File → Open Folder → pick that folder
  3. Click ⚡ to open Claude and ask it to create a CLAUDE.md
  4. Then describe what you want to build — Claude does the rest
Claude will ask you to click Accept a bunch of times. That's just it asking permission before touching your files. You already told it what to do — just say yes.
