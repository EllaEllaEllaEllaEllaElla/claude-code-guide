Two tools you'll hear about
🖥️

VS Code — your workspace

VS Code is a code editor — think of it like Microsoft Word, but for building things. It's where you'll open your projects, see your files, and talk to Claude. This type of app is called an IDE (Integrated Development Environment) — a fancy name for "the place where you work on code."

⬛

Terminal — the command line

Terminal is a window where you type instructions directly to your computer, no clicking required. It looks intimidating but you'll only use it twice in this whole guide — to install Claude Code and confirm it worked. After that, you do everything from inside VS Code.

1

Install VS Code

  1. Go to code.visualstudio.com and download VS Code for Mac
  2. Open the download — drag Visual Studio Code into your Applications folder
  3. Open it (right-click → Open the first time)
2

Install Claude Code

  1. Open Terminal — press Cmd + Space, type Terminal, press Enter. A window with a blinking cursor appears.
  2. Paste this and press Enter:
$ curl -fsSL https://claude.ai/install.sh | bash
  1. Wait for it to finish — type claude --version to confirm it worked
3

Log In

  1. In Terminal, type claude and press Enter
  2. A browser window will open — sign in with your Anthropic account
  3. Done. You won't need to do this again.
Heads up: Claude Code requires a paid plan — Claude Pro at $20/month minimum. Free accounts won't work. Sign up at claude.ai/upgrade.
4

Install the VS Code Extension

  1. Open VS Code and press Cmd + Shift + X
  2. Search for Claude Code
  3. Install the one by Anthropic — double-check the publisher name, there are fakes
  4. Restart VS Code if it asks
5

Set Up Your Projects Folder

  1. Go to your Desktop and create a new folder called projects
  2. Every time you start something new, make a new folder inside it — like projects/birthday-card or projects/my-website. One project, one folder.
  3. To start working: open VS Code, go to File → Open Folder, and pick that project's folder
  4. Claude can now see everything inside and get to work
Think of opening a folder like handing Claude the right notebook. Always open the right one before you start.
6

Your First Files

  1. Create a new folder inside projects for your first project and open it in VS Code
  2. Click the ⚡ spark icon to open Claude and type:
create a CLAUDE.md file that describes this project
  1. Claude will make the file and fill it in — you don't need to do anything else
  2. Do this at the start of every new project
Claude reads this file automatically every time you open the project. It's how Claude remembers what you're building. Think of it as a sticky note you leave for Claude.
  1. Inside projects, create a folder called ai-progress and open it in VS Code
  2. Click the ⚡ spark icon and type:
create a file called progress.md where I can track what I've learned
  1. Claude will create a .md (markdown) file — a simple text format, like a notes file that any app can read
  2. Write your first entry: what you set up today
Markdown is just plain text with light formatting — # makes a heading, - makes a bullet. That's most of what you need.
7

Build Something

  1. Create a new folder inside projects, name it after your project, and open it in VS Code
  2. Click ⚡, ask Claude to create a CLAUDE.md, then describe what you want to build
  3. Claude will start working — it may ask you to click Accept a few times along the way. That's just Claude asking permission before it touches your files. Most of the time, just click yes.
