Hi Doug! Let's get you set up.

8 steps. No experience needed.

1

Install VS Code

  1. Go to code.visualstudio.com and download VS Code for Mac
  2. Open the download — drag Visual Studio Code into your Applications folder
  3. Open it (right-click → Open the first time)
2

Install Claude Code

  1. Open Terminal — it's an app that comes with every Mac. Press Cmd + Space to open Spotlight (the search bar), type Terminal, and press Enter. A window with a blinking cursor will appear. That's it.
  2. Paste this and press Enter:
$ curl -fsSL https://claude.ai/install.sh | bash
  1. Wait for it to finish — type claude --version to confirm
3

Log In

  1. In Terminal, type claude and press Enter
  2. A browser window will open — sign in with your Anthropic account
  3. That's it. You won't need to do this again.
Heads up: Claude Code requires a paid plan — Claude Pro at $20/month minimum. Free accounts won't work. Sign up at claude.ai/upgrade.
4

Install the VS Code Extension

  1. Open VS Code and press Cmd + Shift + X
  2. Search for Claude Code
  3. Install the one by Anthropic — double-check the publisher name, there are fakes
  4. Restart VS Code if it asks
5

Set Up Your Projects Folder

  1. Go to your Desktop and create a new folder called projects
  2. Every time you start something new, create a new folder inside projects for it — like projects/birthday-card or projects/my-website. One project, one folder.
  3. Open VS Code, go to File → Open Folder, and select the folder you just made
  4. Claude can now see everything inside that folder and work with it
Think of opening a folder like telling Claude "this is the project we're working on." Always open the right folder before you start.
6

Make Your First Markdown File

  1. Inside projects, create a new folder called ai-progress and open it in VS Code
  2. Click the ⚡ spark icon to open Claude, then type:
create a file called progress.md where I can track what I've learned
  1. Claude will create a .md file for you — a markdown file. It's a simple text format that any app can read, great for notes and lists.
  2. Add your first entry: what you set up today
Markdown files are just text with light formatting — like using # for a heading or - for a bullet. You'll see them everywhere in coding projects.
7

Set Up CLAUDE.md

  1. For any project you work on, ask Claude to create a CLAUDE.md file in that folder
  2. Type something like:
create a CLAUDE.md file that describes this project so you always remember what we're building
  1. Claude reads this file automatically every time you open the project — it's how Claude remembers what you're working on, what style you want, and what not to touch
  2. Do this at the start of every new project
This is the #1 thing most beginners skip. The best Claude Code users always have a CLAUDE.md. It's the difference between Claude that forgets everything and Claude that feels like it actually knows your project.
8

Build Something

  1. Create a new folder inside projects for your first project and open it in VS Code
  2. Click the ⚡ spark icon and ask Claude to create a CLAUDE.md first
  3. Then just describe what you want — Claude does the rest

You're all set, Doug.

make a beautiful late birthday card
