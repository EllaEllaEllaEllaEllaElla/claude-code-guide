Hi Doug! Before we start —

Here are the tools and terms you'll hear about. No need to memorize anything, just skim so nothing catches you off guard.

The tools
🤖Claude Code
The AI you'll be talking to. It reads your files, writes code, and builds things — all from plain English instructions. Made by Anthropic.
🖥️VS Code
Your workspace. A free app where you open projects, see your files, and chat with Claude. This type of app is called an IDE (Integrated Development Environment) — basically "the place where you work on code."
📝Obsidian
A note-taking app. Great for keeping track of what you've learned, ideas, and progress. Stores everything as simple text files on your computer.
🐙GitHub
Where your code lives online. Think of it like Google Drive but for projects — it backs up your work and keeps a history of every change you've made.
⬛Terminal
A window where you type instructions directly to your Mac's operating system — no clicking, just text. You'll use it once to install Claude Code. After that, Claude Code talks to the terminal on your behalf: whenever Claude runs a command, installs something, or moves files around, it's doing that through the terminal without you having to touch it.
Words you'll see
📁Folder / Directory
Same thing. A folder on your computer. Every project lives in its own folder.
📄Markdown (.md)
A simple text format used everywhere in coding. Just plain writing, with # for headings and - for bullets. Claude uses it for notes and docs.
🗒️CLAUDE.md
A special file you put in every project. Claude reads it automatically — it's how Claude remembers what you're building so you don't have to re-explain every time.
✅Accept / Approve
Claude asks before changing your files. You'll click this a lot. It's normal — just say yes.

That's all the vocab. Let's install things.
