Install Claude Code

4 steps and you're done.

1

Install VS Code

  1. Go to code.visualstudio.com and download VS Code for Mac
  2. Open the download — drag Visual Studio Code into your Applications folder
  3. Open it (right-click → Open the first time)
2

Install Claude Code

  1. Open Terminal — press Cmd + Space, type Terminal, press Enter
  2. Paste this and press Enter:
$ curl -fsSL https://claude.ai/install.sh | bash
  1. Wait for it to finish — type claude --version to confirm it worked
3

Log In

  1. In Terminal, type claude and press Enter
  2. A browser window will open — sign in with your Anthropic account
  3. Done. You won't need to do this again.
Heads up: Claude Code requires a paid plan — Claude Pro at $20/month minimum. Free accounts won't work. Sign up at claude.ai/upgrade.
4

Install the VS Code Extension

  1. Open VS Code and press Cmd + Shift + X
  2. Search for Claude Code
  3. Install the one by Anthropic — double-check the publisher name, there are fakes
  4. Restart VS Code if it asks

Claude Code is ready.
