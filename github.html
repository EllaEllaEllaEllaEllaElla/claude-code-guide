Create a GitHub Account

Where your code lives online — and how you back it up.

1

Sign Up

  1. Go to github.com and click Sign up
  2. Enter your email, create a password, and pick a username
  3. GitHub will send you a verification code — check your email and enter it
  4. Skip any optional setup questions and click Continue
GitHub is free. You don't need a paid plan for anything in this course.
2

What GitHub Is For

  1. GitHub stores your code online so it's never lost
  2. It keeps a full history of every change you make — like undo, but forever
  3. Later, you'll use it to publish websites and share projects
You don't need to understand everything about GitHub right now. Just having the account is enough to get started.
