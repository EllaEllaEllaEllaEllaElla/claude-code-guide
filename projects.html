Set Up Your Projects Folder

One place for everything you build.

1

Create the Folder

  1. Go to your Desktop and create a new folder called projects
  2. Every time you start something new, make a new folder inside it — like projects/birthday-card or projects/my-website
  3. One project, one folder. Always.
2

Open a Project in VS Code

  1. Open VS Code
  2. Go to File → Open Folder
  3. Select the folder you want to work on
  4. Claude can now see everything inside and get to work
Think of opening a folder like handing Claude the right notebook. Always open the right one before you start.

That's it.

Simple system. You'll use it every single time.
